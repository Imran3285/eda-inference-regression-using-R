Global Development Analysis

Exploring the relationship between GDP per capita and life expectancy
EDA Visualisation Descriptive statistics Gapminder ggplot2 dplyr

Overview

This project uses the Gapminder dataset — 142 countries observed from 1952–2007 — to explore how economic development (GDP per capita) relates to population health outcomes (life expectancy). Analysis proceeds through univariate distributions, bivariate scatter plots, and group-level comparisons by continent.

Reusable summary function

Rather than repeating the same summarise() call for each variable, a helper function accepts any column via tidy evaluation:

summarise_by_continent <- function(data, var) {
  data %>%
    group_by(continent) %>%
    summarise(
      n      = n(),
      mean   = round(mean({{ var }}), 2),
      median = round(median({{ var }}), 2),
      sd     = round(sd({{ var }}), 2),
      iqr    = round(IQR({{ var }}), 2),
      q1     = round(quantile({{ var }}, 0.25), 2),
      q3     = round(quantile({{ var }}, 0.75), 2),
      .groups = "drop"
    )
}

summarise_by_continent(gapminder, lifeExp)
summarise_by_continent(gapminder, gdpPercap)
Finding: Higher GDP per capita is positively associated with longer life expectancy. African nations show the lowest median life expectancy; Oceania the highest. The bimodal density curve reflects a persistent development gap between wealthier and lower-income nations.

Simulation function

The original code used a for loop to build each sampling distribution. This is replaced by a generic, reusable function using replicate() — cleaner and idiomatic R:

simulate_sample_means <- function(population, n, n_sims = 5000) {
  replicate(n_sims, mean(sample(population, n)))
}

set.seed(42)   # reproducibility
sample_means10  <- simulate_sample_means(area, n = 10)
sample_means50  <- simulate_sample_means(area, n = 50)
sample_means100 <- simulate_sample_means(area, n = 100)

Tidy comparison plot

All three distributions are combined into a single tidy data frame, then faceted — one ggplot call instead of three separate par(mfrow) panels:

sim_df <- bind_rows(
  data.frame(means = sample_means10,  sample_size = "n = 10"),
  data.frame(means = sample_means50,  sample_size = "n = 50"),
  data.frame(means = sample_means100, sample_size = "n = 100")
) %>%
  mutate(sample_size = factor(sample_size,
                              levels = c("n = 10", "n = 50", "n = 100")))

ggplot(sim_df, aes(x = means, fill = sample_size)) +
  geom_histogram(bins = 40, colour = "white", show.legend = FALSE) +
  facet_wrap(~ sample_size, ncol = 1, scales = "free_y") +
  scale_fill_manual(values = c("#D7191C", "#FDAE61", "#1A9641")) +
  theme_portfolio()

Standard error verification

The empirical SD of each sampling distribution should approximate the theoretical standard error σ/√n. The table below confirms the simulation is working correctly:

se_summary <- data.frame(
  n              = c(10, 50, 100),
  empirical_sd   = c(sd(sample_means10), sd(sample_means50), sd(sample_means100)),
  theoretical_se = sd(area) / sqrt(c(10, 50, 100))
)
print(round(se_summary, 2))
Finding: As sample size grows from 10 to 100, the sampling distribution narrows and becomes increasingly bell-shaped. The empirical standard deviations closely track the theoretical standard error — a clean, simulation-based proof of the Central Limit Theorem.

Hypothesis Testing

Impact of maternal smoking on infant birth weight
Two-sample t-test Confidence intervals NC births data Statistical inference

Hypotheses

H₀ — null
μ_smoker = μ_nonsmoker  ·  No difference in mean birth weight by smoking status
H₁ — alternative (two-sided, α = 0.05)
μ_smoker ≠ μ_nonsmoker  ·  A difference exists in either direction

Load & clean

nc   <- read.csv("http://bit.ly/31adfCe")
data <- na.omit(nc)   # 1,000 → 998 complete cases

Group summary

Before testing, compute descriptive statistics per group to understand the observed difference in means:

data %>%
  group_by(habit) %>%
  summarise(
    n           = n(),
    mean_weight = round(mean(weight), 3),
    sd_weight   = round(sd(weight), 3),
    .groups     = "drop"
  )

Two-sample t-test

smoker    <- data$weight[data$habit == "smoker"]
nonsmoker <- data$weight[data$habit == "nonsmoker"]

t_result <- t.test(smoker, nonsmoker,
                   alternative = "two.sided",
                   conf.level  = 0.95)
print(t_result)

# Automated interpretation
if (t_result$p.value < 0.05) {
  cat("Reject H0 — significant difference (α = 0.05).\n")
} else {
  cat("Fail to reject H0 — no significant difference detected.\n")
}
Result: Reject H₀ — there is a statistically significant difference in infant birth weight between smoking and non-smoking mothers (α = 0.05). Babies born to smokers weigh less on average.
Finding: The t-test and 95% confidence interval both indicate a significant association between maternal smoking and lower infant birth weight. While statistical significance is clear, the practical magnitude of the effect is also clinically meaningful — even small average differences in birth weight have real health implications.

Reusable diagnostics function

Residual diagnostic plots are wrapped into a single callable function so they can be applied to any model:

plot_diagnostics <- function(model, title_prefix = "") {
  par(mfrow = c(2, 2), mar = c(4, 4, 3, 1))

  plot(model$fitted.values, model$residuals,
       main = paste(title_prefix, "Residuals vs Fitted"),
       xlab = "Fitted", ylab = "Residuals",
       pch = 16, col = "#5E4FA2")
  abline(h = 0, lty = 2, col = "red")

  hist(model$residuals, breaks = 30, col = "#5E4FA2",
       main = paste(title_prefix, "Residuals"))

  qqnorm(model$residuals, pch = 16)
  qqline(model$residuals, col = "red")

  par(mfrow = c(1, 1))
}

plot_diagnostics(model_log, "Log-Income Model —")
