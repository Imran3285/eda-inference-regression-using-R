# Statistical Analysis Portfolio with R

A professional portfolio of four applied statistics projects built in **R**, covering:

- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Sampling Distributions and the Central Limit Theorem
- Hypothesis Testing
- Correlation Analysis
- Multiple Linear Regression
- Residual Diagnostics

This project demonstrates practical statistical analysis skills using real-world datasets and clear visual storytelling through `ggplot2`.

---

## Project Overview

This repository contains four end-to-end statistical analysis projects completed in R:

### 1. Global Development Analysis
**Topic:** Relationship between GDP per Capita and Life Expectancy  
**Dataset:** Gapminder  
**Methods used:**  
- Univariate analysis
- Kernel density plots
- Histograms
- Boxplots
- Grouped descriptive statistics
- Bivariate scatterplots

**Key insight:** Higher GDP per capita is generally associated with higher life expectancy, with substantial variation across continents.

---

### 2. Sampling Distributions & the Central Limit Theorem
**Topic:** Effect of sample size on the sampling distribution of the mean  
**Dataset:** Ames Housing  
**Methods used:**  
- Simulation
- Repeated random sampling
- Sampling distributions
- CLT visualisation
- Standard error interpretation

**Key insight:** As sample size increases, the sampling distribution becomes more normal and less variable, clearly illustrating the Central Limit Theorem.

---

### 3. Hypothesis Testing
**Topic:** Impact of maternal smoking on infant birth weight  
**Dataset:** North Carolina Birth Records  
**Methods used:**  
- Data cleaning
- Group comparison
- Boxplots
- Two-sample Welch t-test
- Confidence interval interpretation

**Key insight:** Infants born to smoking mothers have significantly lower average birth weight than those born to non-smoking mothers.

---

### 4. Correlation & Multiple Regression
**Topic:** Predicting credit card balance  
**Dataset:** ISLR Credit dataset  
**Methods used:**  
- Correlation matrix
- Scatter plots with regression lines
- Multiple linear regression
- Interaction terms
- Log transformation
- Prediction intervals
- Residual diagnostics

**Key insight:** Credit limit is the strongest predictor of balance, while income and age add more limited explanatory power depending on model specification.
